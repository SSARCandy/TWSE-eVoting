# TWSE Auto eVoting (台灣股東會自動投票系統)

這是一個幫助您**全自動**完成台灣股東會電子投票（股東e票通）的桌面應用程式。

## 🌟 這個程式可以做什麼？

每到股東會旺季，為了順利領取各家公司的**股東會紀念品**，通常需要完成電子投票並出示投票完成的截圖證明。如果您（或家人）持有很多家公司的股票，手動一筆一筆登入點擊投票、截圖存證會花費非常大量的時間與精力。這個程式可以幫您：

- **✅ 自動登入與投票**：只要輸入身分證字號，程式會自動登入「股東e票通」網站，找出所有尚未投票的公司，並自動全部投為「贊成」。
- **✅ 自動截圖存證**：投票完成後，會自動為每一家公司進行網頁全畫面截圖，並依據身分證字號與股號，整齊地存放在您指定的資料夾中。
- **✅ 多帳號連續執行**：您可以一次貼上多組身分證字號（例如幫全家人代投），程式會自動登入 A 帳號 -> 投票截圖 -> 登出 -> 接著登入 B 帳號，一鍵全自動跑到完！
- **✅ 智慧補圖**：如果程式執行到一半中斷，或者您之前已經手動投過票但忘記截圖，程式會自動偵測並「只針對缺圖的公司」進行補截圖，不會重複浪費時間。

## ⚠️ 使用前準備 (非常重要)

在開始使用本程式之前，您的電腦必須滿足以下唯一且最重要的條件：

**必須已安裝「券商網路下單憑證」**  
本程式是透過您的證券商數位憑證進行安全登入。請確保您正在使用的這台電腦中，已經安裝好有效的證券憑證（通常由券商軟體產出並匯入電腦系統）。

**手動驗證**  
您可以手動開啟 Chrome 前往 [股東e票通](https://www.stockvote.com.tw/) 網站，輸入身分證字號、憑證種類選擇「券商網路下單憑證」，確認可以正常登入後，再使用本程式。

> **💡 溫馨提示**：  
>目前僅支援「券商網路下單憑證」登入，以後也不會支援任何其他登入形式。

## 📥 如何下載與執行？

您不需要安裝任何複雜的開發環境或程式碼，只要下載打包好的執行檔即可直接使用：

1. 前往本專案的 [Releases 頁面](https://github.com/SSARCandy/TWSE-Auto-eVoting/releases)。
2. 根據您的作業系統下載對應的檔案：
   - **Windows**：下載結尾為 `.exe` 的檔案。下載後直接點擊兩下即可執行，**免安裝**。
   - **Mac**：下載結尾為 `.dmg` 的檔案。下載後打開，將應用程式拖曳到「應用程式 (Applications)」資料夾即可。

## 📖 簡單使用教學

1. 打開 TWSE Auto eVoting 程式。畫面左側是「設定與進度區」，右側是「網頁自動化區」。
2. 在左側的「身分證字號清單」欄位中，輸入您要投票的身分證字號（可以輸入多組，**逗號分隔**）。
3. 點擊「選擇資料夾」，設定您想要用來儲存「投票截圖」的存放位置。
4. 點擊畫面最下方的「🚀 開始執行」按鈕。
5. **放開雙手，去喝杯水！** 程式會在右側網頁自動完成所有點擊、確認對話框、投票與截圖動作。左側的進度條會即時告訴您目前的處理進度。

> **💡 溫馨提示**：  
> 台灣集中保管結算所（股東e票通）的系統維護時間為台灣時間 **每日凌晨 00:00 至 07:00**，請避開此時段執行程式，以免登入失敗。

## 📁 截圖存檔格式與路徑說明

程式會在您指定的出圖資料夾下，**以每個登入的身分證字號建立獨立的子資料夾**，再將每一家公司的截圖存入對應的身分證資料夾中。檔案名稱會包含**身分證字號**、**投票日期**以及**股票代號**，方便您日後整理或查閱。

- **資料夾結構**：`[指定的儲存資料夾] / [身分證字號] /`
- **檔案名稱格式**：`[身分證字號]_[YYYYMMDD]_[股票代號].png`

**範例**：
假設您設定的儲存位置是 `C:\我的投票紀錄`，使用身分證字號 `A123456789` 在 `2024年05月20日` 幫台積電（代號 `2330`）完成投票，截圖檔案的位置與名稱就會是：
`C:\我的投票紀錄\A123456789\A123456789_20240520_2330.png`
